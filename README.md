# FarmerBot
Discord bot made 100% in python for official xfarmerx's discord

## Deployment

### Windows
```python -m venv .venv```  <br>
```.\.venv\Scripts\Activate.ps1```

### Linux
```python3 -m venv .venv```   <br>
```source .venv/bin/activate```

### Docker compose build
```docker compose up -d --build```
